# 游戏说明

## 数字游戏

### 简介
欢迎来到“数字游戏”，这是一个充满策略与智慧的游戏，既考验个人智慧又考验群体博弈。在游戏中，你将面对一个数字谜题，需要通过思考和预判来提交一个介于 1 到 100 之间的数字，使其尽可能地接近目标数字（Winner Number），获得奖励。游戏分为两个阶段，每个阶段都有独特的规则和奖励机制。

### 规则

#### 第一阶段：无分级奖励
- **目标**：赢得奖励。
- **规则**：
  - 每个玩家提交一个介于1到100之间的数字。
  - 所有玩家提交数字后，系统会计算所有数字的平均值，并取其2/3作为Winner Number。
  - 所提交数字最接近Winner Number的玩家获胜，允许多人同时获胜。
  - 奖励固定

#### 第二阶段：设置分级奖励

- **目标**：获胜并取得尽可能高的奖励级别。
- **规则**：
  - 每个玩家提交一个介于1到100之间的数字。
  - 所有玩家提交数字后，系统会计算所有数字的平均值作为Prize，并取其2/3作为Winner Number。
  - 所提交数字最接近Winner Number的玩家获胜，允许多人同时获胜。
  - 获胜的玩家将获得奖励，奖励随着Prize的评级而变化。
- **评级规则**：根据Prize所在的区间评级，级数越大奖励越高。
  - [0,9]：0级
  - [10,19]：1级
  - ...
  - [90,99]：9级

### 说明

认真思考一下哦，这个游戏是一次性的，类似于解谜游戏，在不知道谜底时最好玩。趣味性就在于人与人之间的博弈，尝试思考一下游戏的“本质”。

### 思考

**Q1**：在一阶段的多次试验中，为了获得总体最大效益，玩家们提交的数字的趋势是什么?（emmm趋势这个词一定程度上指引了思考方向）

**Q2**：在一阶段的单次试验中，你在大量初次玩这个游戏的人中，为了尽可能拿到奖励，提交的数字最好落在哪个区间?（为了简化问题，以十个数字为一个区间。eg：[1,10]，[11,20]...）

**Q3**：在二阶段的多次试验中，为了获得总体最大效益，玩家们提交的数字的趋势是什么?

**Q4**对于分级奖励你有无疑问或能否提供一些建议 .（可以不回答；emmm这个问题放在这里也颇有暗示意味）



游戏源自于经典的博弈论游戏——猜三分之二平均值（Guess 2/3 of the Average），用于研究人群的平均理性。有兴趣的朋友可以搜索相关词条找到解释。











## 答案（别着急看 动动你聪明的大脑呗）

**1.**在一阶段的多次试验中 为了获得总体最大效益，玩家们提交的数字应该要越来越小，并最终趋于 1 .  详解请查看 [A1](A1.html)  (超链接请按住Ctrl+鼠标单击)

**2.**在一阶段的单次试验中，你在大量初次玩这个游戏的玩家中为了尽可能拿到奖励，提交的数字最好在10到20之间.  详解请查看 [A2](A2.html)

**3.**在二阶段的多次试验中，为了获得总体最大效益，玩家们提交的数字应该要越来越大.  详解请查看 [A3](A3.html)

**4.**隐藏奖励：设置10级超级大奖 ，条件 prize为100  详解请查看 [A4](A4.html)
